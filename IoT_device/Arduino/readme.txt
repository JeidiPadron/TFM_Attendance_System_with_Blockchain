Instructions:

On NodeMCU can be loaded ONLY one program

The complete project is located on "NodeMCU-1" folder

The rest of programs can be loaded into the development kit
for testing purpouse, just to check individuals functionalities
a) Blink-test is used to check NodeMCU basic functions
b) fingerprint-emptyDB is used to clean all values stored in fingerprint
c) Fingerprint-enroll is used to add new fingerprint, locally
d) NodeMCU-1 is the complete project
e) OLED-example is used to test the OLED screen
f) WiFiScan-test is used to test Wi-Fi function

For more information about how install libraries see
http://www.arduino.cc/en/Guide/Libraries
